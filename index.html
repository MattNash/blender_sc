Blender Shortcuts

This might develop a bit more order as it goes, but to start it is literally all of the shorcuts I have found useful so far.

  • Tab - Switch between object and edit modes
  • Shift + a - Add an object
  • F3 - Bring up the search bar

Move, Scale and Rotate

  • g - to move
  • r - to rotate
  • s - to scale

While the object is moving pressing x,y,or z will snap the movement to the given axis. Holding the middle mouse button will snap to the nearest axis.

  • mmb - to orbit
  • shift + mmb - to pan
  • scroll wheel - zoom

. - snap to an object, select the object and press the . on the number pad

~ - gives you options to view the object from different angles

o - turns proportional editing on and off. - This is the tool that lets you scroll up and down on the size of the brush affecting your selected object.

  • Alt + z - Xray mode
  • Shift + d - duplicate selection
  • Ctrl + l - If you have a point selected this will select the rest of the object
